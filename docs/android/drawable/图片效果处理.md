# 图片效果处理



## 渐变

```xml
<shape xmlns:android="http://schemas.android.com/apk/res/android">
    <gradient
        android:angle="315"
        android:endColor="#000000"
        android:startColor="#FFFFFF" />
</shape>
```

``android:angle``:此属性设置渐变角度，角度需要为45的倍数，余数部分不足45按照45算。默认0为从左到右，90为从下到上。效果是逆时针旋转。
